# Mobile App Enhancement Project

## Project Overview
This project focused on enhancing an existing mobile application to improve user experience, security, and customer engagement. The goal was to deliver high-impact features while balancing business demands with realistic Agile delivery timelines.

## Features Delivered
- **Password Reset Functionality:** Enhanced account security with a user-friendly password reset flow  
- **Order Tracking:** Provided real-time visibility into customer purchases from placement to delivery  
- **Real-Time Push Notifications:** Delivered order status updates and user communication notifications  

## Role & Responsibilities
- **Requirements Analysis:** Gathered and analyzed business and marketing requirements  
- **User Stories & Acceptance Criteria:** Converted requirements into clear Agile user stories  
- **Backlog & Sprint Facilitation:** Facilitated backlog grooming and sprint planning sessions  
- **Cross-Team Collaboration:** Worked with Product, Marketing, and Engineering teams  
- **Scope Management:** Aligned delivery timelines with team capacity and velocity  

## Tools & Methodologies
- Agile / Scrum framework  
- Jira (backlog management)  
- Stakeholder collaboration and requirements gathering  
- Sprint planning & backlog refinement  

## Outcomes
- **Increased Trust & Engagement:** Security and notification improvements boosted user confidence  
- **Reduced Support Load:** Order tracking reduced customer support inquiries  
- **Timely Delivery:** Enhancements delivered on schedule without scope creep
